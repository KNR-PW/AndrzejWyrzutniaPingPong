# AndrzejLauncher

<img width="2160" height="2880" alt="image" src="https://github.com/user-attachments/assets/e1f45c2a-467b-48fc-b143-141e13eae92b" />


Andrzej z wyrzutnią pingpongową. W repozytorium znajdują się pliki .stl do wyrzutni oraz ich odpowiedniki .sldprt. (Niektóre pliki stl nie mają odpowiedników solidworks), oraz kody potrzebne do uruchomienia robota. Żeby wystrzelić należy trzymać trigger aż do wystrzelenia piłki, następnie należy go puścić i odczekać do powrotu sprężyn na pozycję początkową. Należy pamiętać o gumce recepturce, która ma trzymać "driver" i "driver spring mount" razem. 

Kod jest bardzo prowizoryczny, repo zostanie zaktualizowane przeze mnie na VI piknik naukowy na Bemowie. Poniżej załączam obrazy przydatne podczas składania wyrzutni.
